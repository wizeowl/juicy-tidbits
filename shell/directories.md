#Directories

```bash
/bin        # system binaries
/home    # user's home directory
/dev       # device files like block and character devices
/etc        # system configuration
/lib         # shared library and kernel modules
/lib64   
/var        # data wich varies over time
/sbin      # admin and root binaries
/mnt      # mount point or an fs
/tmp     
/proc      # kernal data structure mounted as fs (only linux)
/boot      # contains initramfs and kernel image
/sys        # kernal data structure for different hardware and devices, ..
```
